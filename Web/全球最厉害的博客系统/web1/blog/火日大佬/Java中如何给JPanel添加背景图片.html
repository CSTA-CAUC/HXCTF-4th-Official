Java中如何给JPanel添加背景图片

火日攻天

    用Swing做图形界面的时候发现，JPanel还是不支持直接添加背景图片，这点比C#差多了，即使按照下面添加成功了还是不能实时的看到背景图片，一定要运行起来才能看到，而C#可以直接预览

    不多说上代码，原理是重写paintComponent方法，所以背景图片可以根据panel大小的变化实时的变化

    Panel panel= new javax.swing.JPanel()  
{
    public void paintComponent(Graphics g) {
        ImageIcon icon =new ImageIcon(getClass().getResource("/res/panel.png"));
        // 图片随窗体大小而变化
        g.drawImage(icon.getImage(), 0, 0, panel.getSize().width,panel.getSize().height,panel);
    }
}
