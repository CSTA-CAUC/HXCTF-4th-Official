二、插件的安装

先登陆Package Controller官网：然后在上面复制（这是一段python代码）
复制之后，在sublime text2中按快捷键ctrl + ~，然后将刚复制的代码粘贴到这，按下回车
然后重启sublime text2，重启后，按下快捷键ctrl + shift + p，然后在框内输入install package
稍等片刻，会出来一个插件列表。 然后在里面输入你所要安装的插件
至于快捷键：
Coding（比较喜欢eclipse的快捷键，直接改成了eclipse的）:

操作方面(不断更新中)：
ctrl + shift + p 插件安装
ctrl + p 命令行
#attr 查找属性
:num 就停留在某一行了
@id/fun 查找对应的class和id
ctrl+H 替换
ctrl+D 匹配相同内容，多行游标选择
Ctrl+Shift+[ 折叠代码
Ctrl+Shift+] 展开代码
Ctrl + +/-变大变小
Ctrl+Shift+f 可以查找
Ctrl+J 将多行换成单行
Ctrl+Shift+' 选中标签
Ctrl + D 选中相同的代码
Ctrl + Shift + D 复制整行代码
Ctrl + Shift + A 由内向外扩展
Ctrl + Shift + M 选中{}内的内容 直接在里面输入字母是找文件

插件部分：
  • AdvancedNewFile 快速新建文件
  • Emmet 不解释，快上车
  • SublimeCodeIntel
  • SublimeTmpl
  • BracketHighlighter
  • docblockr
  • cats
  • markdown preview
  • markdown editing
  • nodejs
  • AutoFileName 快速引入资源
  • Insert Filepath 也是快速引入资源 (command+option+I) 感觉就是有点卡
  • SublimeLinter 检查代码错误
  • SublimeLinter-jshint 检查JS代码错误
  • sidebar
  • SideBarFolders 管理文件夹
  • HTML-CSS-JS Prettify: （美化）html、css、js
  • Bracket Highlighter 用于匹配括号，引号和html标签。对于很长的代码很有用。安装好之后，不需要设置插件会自动生效
  • highlighter
  • view in browser
  • ConvertToUTF8
  • Terminal -> 终端（快捷键:Command+Shift+T）
  • Anaconda -> python自动补齐 但需要配置一下，不然有白块，影响coding {"anaconda_linting":false}

Color Highlighter -> 显示颜色（写CSS时相当好用）
