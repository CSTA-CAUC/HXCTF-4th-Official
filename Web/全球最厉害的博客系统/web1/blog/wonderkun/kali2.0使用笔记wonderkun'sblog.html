kali2.0 使用笔记

wonderkun 撰写 无回复

kali 2.0:
0x1.安装kali
1.先将kali 2.0的ios镜像烧录到U盘中，烧录的工具很多的，但是个人觉得还是rufus好用一些。
启动之后是这个样子的。

2.进入系统的bios然后设置从u盘启动，就可以开始安装了。

注意：
安装的过程可能会爆一个错误！
提示说：探测并挂载光驱失败！！
看到这里我就忍不住要骂人了！我是从u盘安装的，再说了老子电脑根本没有光驱好不…………，所以说肯定是系统搞错了，把我的u盘当成光驱挂载了
解决办法：
选择否，不进行光驱探测，进入安装流程的主菜单，选择运行shell：


输入  df -m
果真看到用这么一条 /dev/sda1  / #media 。这个是U盘设备挂载到了/media，导致cd-rom不能被挂载。  
输入一下命令  umount  /media
然后进入安装流程的主界面，然后在重新执行语言选择，就可以安装了！

0x2.kali的简单的配置：
1.修改更新源 ，建议用阿里的 因为比较快

vim  /etc/apt/sources.list   
注释掉之前的源，更换为
#阿里云kali源
deb http://mirrors.aliyun.com/kali kali main non-free contrib
deb-src http://mirrors.aliyun.com/kali kali main non-free contrib
deb http://mirrors.aliyun.com/kali-security kali/updates main contrib non-free
然后  apt-get update &apt-get dist-upgrade  刷新系统！

2.安装中文的输入法

apt-get install fcitx-googlepinyin
init 6 #重启一下，然后进入系统，ctrl+space就可以进行输入法的切换，如果想改变默认的配置，可以用Fcitx配置    
#直接在桌面搜索窗口输入fcitx就可以看到！

3.安装flash 

apt-get install flashplugin-nonfree #安装flash  palyer
update-flashplugin-nonfree   --install  #并且安装更新

4.安装一个可以多窗口的终端

apt-get  install terminator

6.安装linux的shadowssocks的客户端

7.如果还想在kali里面玩下windows，那就需要安装vmware workstation 或者virtualbox了，但是在linux上我更推荐玩virtualbox
安装virtualvbox, 难道不是 apt-get install virtualbox吗？ 当然 不是这么简单了，否则我就不说了。
如果你这样安装了之后就会报以下错误

Kernel driver not installed (rc=-1908)
The VirtualBox Linux kernel driver (vboxdrv) is either not loaded or there is a permission problem with /dev/vboxdrv. Please reinstall the kernel module by executing
‘/etc/init.d/vboxdrv setup’
as root. If it is available in your distribution, you should install the DKMS package first. This package keeps track of Linux kernel changes and recompiles the vboxdrv kernel module if necessary.

为啥会出错呢，因为你的dkms没安装,要安装dkms才可以

sudo apt-get install    virtualbox  virtualbox-dkms  
apt-get install linux-headers-$(uname -r)  #安装一些必要的头文件
