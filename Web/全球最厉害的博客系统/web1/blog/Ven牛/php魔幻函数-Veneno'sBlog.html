php魔幻函数

一直想把平常见的一些有意思的Tips总结下:)

php魔幻函数

1。__construct()
实例化对象时被调用，
当__construct和以类名为函数名的函数同时存在时，__construct将被调用，另一个不被调用。

代码参考：
<?php
class myName{
function __construct($myName){    //连续两个下划线(可以试试没有下划线时的区别:)
  echo ("我的名字是：$myName<br>");
}
}
$name1 = new myName("Veneno");
?>

2。__destruct()
当删除一个对象或对象操作终止时被调用。

代码示例：
<?php
class A{
function __destruct(){
echo 'Veneno<br/>';
}
}
$a = new A();
//unset($a);
echo 'Hello<br/>';
?>

3。__call()
对象调用某个方法，若方法存在，则直接调用;
若不存在，则会去调用__call函数。

代码示例：
<?php
class A{
public function __call($name,$arguments){
echo 'hi<br/>';
}
function a(){
echo 'Veneno<br/>';
}
}
$a = new A();
$a->a();
?>

4。__get()
读取一个对象的属性时，若属性存在，则直接返回属性值;
若不存在，则会调用__get函数。

代码示例：
<?php 
class Person 
{ 
//下面是人的成员属性，都是封装的私有成员 
private $name; //人的名子 
//__get()方法用来获取私有属性 
private function __get($property_name) 
{ 
echo "在直接获取私有属性值的时候，自动调用了这个__get()方法<br>"; 
if(isset($this->$property_name)) 
{ 
return($this->$property_name); 
} 
else 
{ 
return(NULL); 
} 
} 
//__set()方法用来设置私有属性 
private function __set($property_name, $value) 
{ 
echo "在直接设置私有属性值的时候，自动调用了这个__set()方法为私有属性赋值<br>"; 
$this->$property_name = $value; 
} 
} 
$p1=newPerson(); 
//直接为私有属性赋值的操作，会自动调用__set()方法进行赋值 
$p1->name="张三"; 
//直接获取私有属性的值，会自动调用__get()方法，返回成员属性的值 
echo "姓名：".$p1->name."<br>"; 
?>

5。__set()
设置一个对象的属性时，若属性存在，则直接赋值;
若不存在，则会调用__set函数。

6。__toString()
打印一个对象的时被调用。如echo $obj;或print $obj;

代码示例：
<?php
class A{
function __toString(){
echo 'Hi';
}    
function a(){
echo 'Veneno<br/>';
}
}
$a = new A();
//echo $a;
?>

7。__sleep()
serialize之前被调用。

代码示例：
<?php
class A{
function __sleep(){
echo 'Hi';
}    
function a(){
echo 'Veneno<br/>';
}
}
$c = new A();
//$b = serialize($c);
?>

8。__wakeup()
unserialize时被调用，做些对象的初始化工作。

代码示例：
<?php
class A{
function __wakeup(){
echo 'Hello';
}    
}
$c = new A();
$v=serialize($c);
$d=unserialize($v);
?>
