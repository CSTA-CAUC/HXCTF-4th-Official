    在学校一直就想放假了装一个BT5。现在总算放假了，昨天下了个BT5r3虚拟机装上了，第一步先汉化。

    在BT5命令行下输入 apt-get install language-support-zh language-pack-zh 安装中文语言包。等完成了再输入apt-get install language-selector 安装语言包支持。这两步应该没什么问题，系统都帮你自动下好安好。

    再重启(命令行下输入reboot)应该就OK了。

    最后，下个中文输入法，ibus. 命令行下输入 apt-get install ibus-pinyin 就OK了。安装好了按Ctrl + 空格 切换到该输入法。

    

    再看firefox，此时还是英文状态，十分不爽。

    好吧，在firefox下打开 http://stage.mozilla.org/pub/mozilla.org/firefox/releases/ ，在其中选择你的FireFox相对应的版本的目录，比如我的是14.0.1 。再选择linux-i686,再选择xpi (注意不是zh-CN),进入xpi这个文件夹了再选择zh-CN.xpi 然后浏览器会安装，按照他告诉你的走(你不要告诉我这点英文你看不懂)。他会告诉你firefox重启后就能安装xpi了，但是重启了后发现什么动作也没有。

    其实还有工作要做：

    在地址栏里输入about:config 进入设置页面，在里面找general.useragent.locale,把它的值改成zh-CN就行了。再重启火狐试试！！

    汉化到此为止，我也还没有深入了解过backtrack，这个寒假可以尽情放松放松学点有意思的东西了。
