2017年,新的开始

作者：albertchang 发布时间：January 1, 2017 分类：杂记

2016年就这样悄然的过去了，回首2016，走的磕磕绊绊的，各种成绩也不尽如人意，因为很多原因吧，博客也荒废了。

  1. 最开始使用conoha的vps，因为一些原因被封了。之后换了很多的vps，从linode到星光互联，再到vultr，博客也从php换成了python和nodojs，也尝试着自己写了几次博客，但是都不是太满意。兜兜转转又用回了typecho的博客系统。
  2. 很久没有写博客了，借着新年的机会，从开一个博客。记录些自己的经历和变化，也记录着自己的人生路上的行程。
  3. 首先立几个flag吧：
    • 好好地陪女朋友走下去。已经走过了4个新年的我们，要继续好好的走下去。
    • 熟悉nodejs，完善之前自己写的那个不太成熟的blog，并在2017年年底之前替换掉typecho。
    • 研究web，多做渗透，完善自己的渗透思路，不要把自己渗透的思路局限死。 - 踏实下来做做cms的源码审计，目标是至少3个以上的0day。不在把自己的思路局限在php，还有jsp，python。
    • 拼尽全力打完最后半年的每一次CTF比赛，做到自己的问心无愧。
    • 好好研究docker，争取把批量化的docker管理控制平台写完。这也是我从2016年就开始做的事情，却一直拖拖拉拉的没有做好。
    • 踏实下来，认真总结一下以前的各种经验。
  4. 2017年，走的更好！
