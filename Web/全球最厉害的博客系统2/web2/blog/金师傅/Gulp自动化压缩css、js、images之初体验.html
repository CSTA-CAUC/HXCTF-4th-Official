Gulp自动化压缩css、js、images之初体验
  • Kylin King
  • 二月 21 2016
  • 前端

原来看很多大神提到Grunt、 Gulp前端自动化，一直没去尝试。~

今天算是见识到了自动化好处的冰山一角了，用Gulp进行自动化压缩css和js。

使用方法
一、安装gulp

全局安装一遍:

npm install gulp -g

然后到项目的根目录里,再装一遍

npm install gulp --save-dev(根目录下要有*package.json*文件)
二、安装Gulp插件

选择性的安装啦

~ * 自动添加css前缀（gulp-autoprefixer） * 压缩css（gulp-minify-css） * js代码校验（gulp-jshint） * 合并js文件（gulp-concat） * 压缩js代码（gulp-uglify） * 压缩图片（gulp-imagemin） * 自动刷新页面（gulp-livereload） * 图片缓存，只有图片替换了才压缩（gulp-cache） * 更改提醒（gulp-notify） * 清除文件（del） 
$ npm install gulp-autoprefixer gulp-minify-css gulp-jshint gulp-concat gulp-uglify gulp-imagemin gulp-notify gulp-rename gulp-livereload gulp-cache gulp-clean gulp-changed-in-place del --save-dev
三、根目录下新建gulpfile.js

在文件中写入如下代码：

//引入相关模块
var gulp = require('gulp');  
var minifycss = require('gulp-minify-css');  
var uglify = require('gulp-uglify');  
var concat = require('gulp-concat');  
var del = require('del');

gulp.task('homecss',function(){  //压缩css,其中homecss可以自定义名称  
    return gulp.src('./Apps/Home/View/default/css/*.css')  //未压缩css
    .pipe(minifycss())  //执行压缩方法
    .pipe(gulp.dest('./Apps/Home/View/default/min/css'))  //压缩后的css
});  

gulp.task('homejs',function(){  //压缩js，其中homejs可以自定义名称  
    return gulp.src('./Apps/Home/View/default/js/*.js')  //未压缩js
    .pipe(uglify())  //执行压缩方法
    .pipe(gulp.dest('./Apps/Home/View/default/min/js'))  //压缩后的js
});  

gulp.task('default',function() {  //当在命令行中输入gulp时，执行的任务  
    gulp.start('homecss','homejs');

    gulp.watch('./Apps/Home/View/default/css/*.css',function(){  //监控
        gulp.run('css');  
    });  

    gulp.watch('./Apps/Home/View/default/js/*.js',function(){  //监控
        gulp.run('minifyjs');  
    });
});

才开始时，是有用清除文件的方法的

gulp.task('clean', function(cb) {  
    del(['./Apps/Home/View/default/min/css', './Apps/Home/View/default/min/js'], cb)
});

结果咋折腾都提示文件已存在，无法覆盖先前的文件。所以只能写个小bat脚本，先执行系统的删除命令，再执行gulp。

四、编写bat脚本(可选)
@echo off
:: gulp自动化脚本
SET JSFOLDER=C:\Users\Administrator\Desktop\formini  
@echo 正在压缩...
del d:\wamp\www\zhaofudao\Apps\Home\View\default\min /q /s  
del d:\wamp\www\zhaofudao\Apps\Wap\View\default\min /q /s  
gulp  
echo 已全部压缩完毕!  
pause & exit

然后将bat文件添加到system32中，以后要压缩zfd项目时，直接win+R ,输入zfdmin 就自动压缩到新的文件夹中，然后向服务器传时，直接传新文件夹中的。方便的不要不要的~~~^^ ^^。
