React-Native之初体验
  • Kylin King
  • 五月 19 2016
  • 前端

React-Native是Facebook的一个开源项目，话说用JS开发的代码抽象成原生UI库，直接可以在IOS和Android上直接运行。

首先说下安装吧：
  • 安装最新版的XCode
  • 安装HomeBlewruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  • 安装Node.js(直接到官网下载：NodeJS)
  • 安装WatchMan(用于修改Source文件):brew install watchman
  • 安装Flow(JS静态类型检查器):brew install flow
  • 安装React Native CLI:sudo npm install -g react-native和sudo npm install -g react-native-cli
创建一个HelloWorld项目吧
  • 打开终端，运行react-native init HelloWorld
  • 然后进入到这个项目文件夹里，运行一下react-native run-ios
