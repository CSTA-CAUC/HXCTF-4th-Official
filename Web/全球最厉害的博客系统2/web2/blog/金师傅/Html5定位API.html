Html5定位API
  • Kylin King
  • 十二月 20 2015
  • 前端

Html5想必大家早有耳闻，其实我个人觉得吧，真正意义上Html5的酷炫，来源于CSS3和JavaScript，就html5定位技术，就是javascript新的api属性带来的。见如下code

<!doctype html>  
<html lang="en">  
<head>  
  <meta charset="UTF-8">
  <title>Html5定位</title>
  <script>
  window.onload = function(){
   window.navigator.geolocation.getCurrentPosition(function(position){
       alert(position.coords.latitude);//获取纬度
       alert(position.coords.longitude);//获取经度
   })
}
  </script>
</head>  
<body>

</body>  
</html>


说明：定位在手机上使用，电脑上不一定能定成功(因为我有一次在电脑上alert出了自己的经纬度,吓坏宝宝了)

演示地址

只要获得了经纬度，便可以结合百度地图/高德地图，进行更进一步的开发。这里我用的百度api，直接就可以做出一个地图定位。所以还是挺强大的。 

//省略了地图接入的过程
function success (position){  
    var la = position.coords.latitude;
    var lo = position.coords.longitude;
    // 百度地图API功能
    var map = new BMap.Map("map1");            // 创建Map实例
    map.centerAndZoom(new BMap.Point(""+la+"", ""+lo+""), 11);
    var local = new BMap.LocalSearch(map, {
        renderOptions:{map: map, autoViewport:true}
    });
    local.searchNearby("小吃", "前门");
}
