Proxychain-ng + Shadowsocks实现Mac终端下代理
  • Kylin King
  • 五月 22 2017
  • 前端 服务器

shadowsocks 能实现浏览器端翻越 GWF，但是仅靠 shadowsocks 是无法终端 Terminal 越过 GWF ，而对于开发而言，终端的重要性不言而喻，常用的 git clone 、 brew install 、 npm install 在终端运行的命令在不设置源镜像源的情况下，速度非常之慢。

安装Shadowsocks

客户端下载

安装完之后，在 /Users/xxx/.ShadowsocksX-NG/gfwlist.js 看下监听的哪个端口。

安装Proxychain-ng

使用 brew 安装(若未安装，见Homebrew安装): 

$ brew install proxychains-ng

用 vim 配置代理端口：

$ vim /usr/local/etc/proxychains.conf

在最后一行，注释掉 socks4，添加 shadowsocks 的代理端口

* 注意：配置完后，要重新打开一下 终端 ，代理才能生效！ *

使用

在命令前( git 、 brew 等 )加上 proxychains4 ，例如 clone 一下 rax 项目。 

$ proxychains4 git clone https://github.com/alibaba/rax.git

速度是杠杠的 ~
